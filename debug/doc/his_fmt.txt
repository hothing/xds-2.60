        Формат файла, используемый утилитой his.

1. Используется утилитой his для показа посмертного (в момент развала)
   состояния стека вызовов программы.

2. Файл генерируется RTS.

3. Формат файла:

   - файл является обычным текстовым файлом

   - первая строка содержит полное имя программы, например:
     d:\xd\xd_win32\his.exe

   - вторая строка содержит причину завершения программы, например:
     #RTS: unhandled exception #21: USER BREAK

   - далее следуют строки следующего вида:

     <Offset> <Object> <Component>

     где: Offset    - смещение относительно сегмента
          Object    - номер сегмента (обьекта) компоненты, начиная с нуля
          Component - путь до компоненты

     например, так:
           
     000005A3 00000001 D:\XD\XD_WIN32\di_tls.dll
     00000785 00000000 D:\XD\XD_WIN32\dry.exe
