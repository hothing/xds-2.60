Компонента xd_utl: разнообразные полезные инструменты

  src\xd_utl\xd_utl.mod

    Главный модуль компоненты xd_utl - вспомогательные модули
    Компонента содержит разнообразные полезные инструменты
    
    Проверяет срок работы пробной версии (TRIALVERSION)
    

  src\xd_utl\Dll.def

    Поддержка использования библиотек динамического связывания.
    

  src\xd_utl\File.def

    Функции для работы с файлами: преобразования имен файлов,
    расширений; модификации и сравнения времени создания
    

  src\xd_utl\MsgNo.def

    Определение номеров сообщений отладчика.
    

  src\xd_utl\Options.def

    Набор переменных, определяющих режим работы отладчика
    

  src\xd_utl\Protocol.def

    Модуль управления созданием протокола отладки и выдачей сообщений
    
    Данный модуль берет список сообщений из заданого файла.
    Поиск файла осуществляеться по red-файлу системы, если
    не найден - поиск продолжаеться в "домашней", т.е.
    откуда запущен.
    

  src\xd_utl\RedFile.def

    Модуль, предоставляющий функции для работы с перенаправляющим
    файлом системы. Структура списков путей поиска создается на
    основе указанного файла.
    

  src\xd_utl\Sort.def

    Процедуры для сортировки и поиска информации.
    

  src\xd_utl\Texts.def

    Модуль предоставляет абстрактное понятие
    текста и набор процедур для работы с ним
    

  src\xd_utl\xStr.def

    Различные процедуры для работы со строками.
